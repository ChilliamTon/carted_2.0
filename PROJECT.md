# Wishlist Central

Web-first wishlist aggregation app with price tracking.

## Stack
- Frontend: React + Vite + TypeScript + TailwindCSS
- Backend: Supabase (PostgreSQL + Auth + Edge Functions)
- Scraping: Puppeteer/Cheerio
- Testing: Vitest + Playwright

## MVP Features
1. Browser extension to save products
2. Lists + folders organization
3. Price tracking
4. Availability tracking
